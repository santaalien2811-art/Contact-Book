## Contact Book

A simple Contact Book application built with Python that allows users to manage contacts from the command line.

## Features
- Add contacts
- View all contacts
- Search contacts by name or phone number
- Edit existing contacts
- Delete contacts
- Count total contacts
- Prevent duplicate contacts
- Save contacts to a file
- Load contacts automatically when the program starts
- Technologies Used
- Python 3
- Dictionaries
- Lists
- Functions
- File Handling
## How It Works

Each contact contains:

- Name
- Phone Number
- Email

Contacts are stored in a text file so they remain available after closing the program.
